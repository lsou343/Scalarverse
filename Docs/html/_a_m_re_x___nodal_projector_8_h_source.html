10//
11//
12// *************************** DEFAULT MODE ***************************
13//
14// Solves
15//
16// div(sigma*grad(phi)) = div(vel) + S_nd + S_cc
17//
18// and performs the projection
19//
20// vel = vel - sigma * grad(phi)
21//
22// vel, sigma, and S_cc are cell-centered variables, while
23// phi and S_nd are nodal-centered variables.
24//
25// *************************** CUSTOM MODE ***************************
26//
27// Solves
28//
29// div(sigma*grad(phi)) = rhs
30//
31// and performs the projection
32//
33// vel = vel - (sigma/alpha) * grad(phi)
34//
35// alpha is a cell-centered variable, while rhs is nodal-centered.
36//
37// In this mode, the user provides rhs and alpha
38//
39// By default alpha is assumed to be 1. Use setAlpha to change the default.
40//
41// Use setCustomRHS to provide a custom RHS, else "div(vel) + S_nd + S_cc"
42// is used.
43//
44// Example: rhs = div(alpha*vel)
45//
