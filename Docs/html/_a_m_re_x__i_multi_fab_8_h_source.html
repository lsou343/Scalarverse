14/*
15* A Collection of IArrayBoxes
16*
17* The iMultiFab class is publically derived from the
18* FabArray<int,IArrayBox> class. It is a collection (stored as an array) of
19* IArrayBoxes useful for storing integer data on a domain defined by
20* a union of rectangular regions embedded in a uniform index space. The
21* iMultiFab class extends the function of the underlying FabArray class just
22* as the IArrayBox class extends the funtion of BaseFab<int>. Additional
23* member functions are defined for I/O and simple arithmetic operations on
24* these aggregate objects.
