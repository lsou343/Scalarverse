92// perform a (inter-level) restriction on vector id_f of the fine level and stores the result in vector id_c on the coarse level
93// restrictionType specifies whether this is either cell-averaged restriction, or one of three face-averaged restrictions
94// piecewise constant restriction requires neither a ghost zone exchange nor a boundary condition
95// This is a rather bulk synchronous implementation which packs all MPI buffers before initiating any sends
96// Similarly, it waits for all remote data before copying any into local boxes.
97// It does however attempt to overlap local restriction with MPI
