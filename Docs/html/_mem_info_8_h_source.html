2 MemInfo: Utility class for checking/logging

7 NOTE THAT THIS CODE IS MACHINE/OS DEPENDENT!

11 Usage:
12 0) To get class instance (singleton class!):

14 MemInfo* mInfo=MemInfo::GetInstance();

16 Optional: to specify file for log summaries

19 mInfo->Init("filename");
20
21 1) To check how many nodes have less than

23

25
26 2) To write one line summary to mem_info.log

31 mInfo->LogSummary(char* info);

33 3) To write full memory info for all MPI ranks to

35
36 mInfo->PrintAll(FILE* fname);

38 4) To get available/free and total memory for "my"
