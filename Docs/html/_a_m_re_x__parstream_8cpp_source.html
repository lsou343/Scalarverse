12// In parallel, each proc sends the stream to a separate file,
13// distinguished by having the proc num in the filename.
14// In serial, the stream goes to std::cout.
15//
16// Since the pout() function gets used a lot, it should be as

19// The other feature is that the application can choose the base filename
